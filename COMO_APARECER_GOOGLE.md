# Como Fazer o Site Aparecer no Google

## ⚠️ IMPORTANTE: O Google NÃO indexa automaticamente!

Mesmo com todo o SEO configurado, você precisa **submeter o site manualmente** ao Google. Isso pode levar de alguns dias a algumas semanas.

---

## 📋 Passo a Passo para Indexação no Google

### 1. **Google Search Console** (OBRIGATÓRIO)

1. Acesse: https://search.google.com/search-console
2. Faça login com uma conta Google
3. Clique em **"Adicionar propriedade"** ou **"Add Property"**
4. Escolha **"Prefixo do domínio"** e digite: `lonclinic.com`
5. Siga as instruções para verificar a propriedade:
   - **Verificação por DNS (Recomendado):** Veja o arquivo `VERIFICAR_DNS_GOOGLE.md` para instruções detalhadas
   - **Opção alternativa:** Adicione um arquivo HTML no servidor

### 2. **Submeter o Sitemap**

Após verificar a propriedade:

1. No Google Search Console, vá em **"Sitemaps"** (no menu lateral)
2. Digite: `sitemap.xml`
3. Clique em **"Enviar"** ou **"Submit"**
4. Aguarde alguns minutos e verifique se foi processado

### 3. **Solicitar Indexação Manual** (Recomendado)

Para acelerar o processo:

1. No Google Search Console, vá em **"Inspeção de URL"** ou **"URL Inspection"**
2. Digite: `https://lonclinic.com/`
3. Clique em **"Solicitar indexação"** ou **"Request Indexing"**
4. Repita para:
   - `https://lonclinic.com/travel-clinic`
   - `https://lonclinic.com/book-consultation`

### 4. **Verificar se está Funcionando**

Após alguns dias, verifique:

1. No Google Search Console → **"Cobertura"** ou **"Coverage"**
   - Veja quantas páginas foram indexadas
   - Corrija erros se houver

2. Pesquise no Google:
   - `site:lonclinic.com`
   - Deve mostrar todas as páginas indexadas

## 🚀 Dicas para Acelerar a Indexação

### 1. **Links Externos**
- Compartilhe o site em redes sociais
- Peça para outros sites linkarem para você
- Adicione o link em perfis de negócios (Google My Business, etc.)

### 2. **Google My Business** (Recomendado)
1. Acesse: https://www.google.com/business/
2. Crie um perfil para "LON Clinic"
3. Adicione endereço, telefone, horários
4. Isso ajuda muito para buscas locais

### 3. **Conteúdo Fresco**
- Atualize o site regularmente
- Adicione blog posts sobre medicina de viagem
- Google prefere sites com conteúdo atualizado

### 4. **Velocidade do Site**
- Certifique-se de que o site carrega rápido
- Use imagens otimizadas (WebP)
- Minimize JavaScript e CSS

---

## ⏱️ Tempo Esperado

- **Primeira indexação:** 3-7 dias (após submeter)
- **Aparecer em buscas:** 1-4 semanas
- **Ranking melhorado:** 1-3 meses (com otimizações contínuas)

## ❌ Problemas Comuns

### "Site não aparece no Google"
- ✅ Verificou no Google Search Console?
- ✅ Submeteu o sitemap?
- ✅ Solicitou indexação manual?
- ⏱️ Aguardou pelo menos 1 semana?

### "Páginas não indexadas"
- Verifique erros no Google Search Console
- Certifique-se de que `robots.txt` permite indexação
- Verifique se há erros 404 ou 500

### "Aparece mas em posição ruim"
- Isso é normal! Melhore com:
  - Mais conteúdo relevante
  - Links externos (backlinks)
  - Otimização contínua
  - Google My Business

---

## 📞 Suporte

Se após 2 semanas o site ainda não aparecer:
1. Verifique o Google Search Console para erros
2. Certifique-se de que o servidor está acessível
3. Verifique se não há bloqueios no robots.txt
